## 4조 커피자판기 프로젝트

본격 커피의 소중함을 느껴보는 자판기 프로젝트

### 프로젝트 기능
* 개쩌는 커피뽑기
* 개쩌는 거스름돈 계산기능
* 개쩌는 투입 금액 총액 계산 및 피드백 기능
* 개쩌는 현실 현금 시스템

### 프로젝트 수정시 참고사항
* 본 프로젝트는 intellij 프로젝트로 이외의 환경에선 동작을 보장하지 않습니다!
* JDK 17 이상 사용을 권장합니다.
* 본 자판기는 커피 회수여부를 판단하는 센서가 없습니다! 당신이 커피 회수 안하시고 커피 하나더 시켜서 커피 흘러 넘치는일은 없도록 하십쇼.
